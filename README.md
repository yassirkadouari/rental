# Car Rental App 

## Description
Application de gestion de location de voitures développée en C# (.NET).
Elle permet de gérer les voitures, les clients et les contrats de location.

## Fonctionnalités
- Authentification des utilisateurs
- Gestion des voitures
- Gestion des clients
- Création des contrats de location
- Génération de contrats PDF

## Technologies utilisées
- C# / .NET
- ASP.NET (Web)
- WPF (Desktop)
- SQL Server

## Architecture
Architecture en couches :
- Core
- Data
- Services
- Web
- Desktop
